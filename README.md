#  Unity Multiplayer Prototype

## Описание
Простой мультиплеерный прототип на Photon PUN 2.
Игроки подключаются к одной комнате и видят друг друга.

##  Функционал
- Подключение к серверу
- Автоматическое создание/вход в комнату
- Спавн игрока
- Синхронизация движения
- Прыжок

##  Как запустить
1. Открыть проект в Unity
2. Нажать Play
3. Запустить второй клиент (Build)
4. Либо запуск двух Exe билдов
5. Игроки автоматически подключатся
